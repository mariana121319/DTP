# MitM Lab: ARP & DNS Spoofing — Documentación Técnica

## Introducción
En este documento se detallará cómo realizar ataques de spoofing utilizando ARP y DNS.

## Requisitos
- Herramientas necesarias: Packet Tracer, Kali Linux, etc.
- Conocimientos previos sobre redes y protocolos de comunicación.

## Pasos a seguir
1. **Configuración de la red:** Ataca los dispositivos de la red configurando sus IP y MAC.
2. **ARP Spoofing:** Utiliza herramientas como 'arpspoof' para engañar a la red.
3. **DNS Spoofing:** Redirige el tráfico DNS a través de 'dnsspoof'.
